Two main paths

You usually only need one of these at a time.

Path A
IDS foundation
Clean IDS once, then use it as the base for any climate dataset.
->
Path A
Climate branch
Choose TerraClimate, PRISM, and/or WorldClim depending on coverage and resolution needs.
Path B
FIA pipeline
Download FIA tables, build state extracts, then compile national plot summaries.
->
Optional
FIA site climate
Add TerraClimate at FIA plot locations only if you need site climate.
