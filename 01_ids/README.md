# IDS — Insect and Disease Detection Survey

**Source:** USDA Forest Service, Forest Health Protection
**URL:** https://www.fs.usda.gov/science-technology/data-tools-products/fhp-mapping-reporting/detection-surveys
**Download date:** 2025-01-29

Annual aerial survey data recording forest insect and disease damage across all USFS regions. Polygons represent observed areas of tree mortality, defoliation, and other damage.

| | |
|---|---|
| **Format** | Geodatabase (.gdb), 10 regional files → GeoPackage |
| **Spatial coverage** | CONUS, Alaska, Hawaii (USFS Regions R1–R10) |
| **Temporal coverage** | 1997–2024 |
| **CRS (raw)** | Mixed (Albers CONUS, Alaska Albers, Hawaii Albers) |
| **CRS (output)** | EPSG:4326 (WGS84) |

## Quick Start

Run scripts in order from the repo root:

```bash
Rscript 01_ids/scripts/01_download_ids.R              # Download 10 regional geodatabases (~1.6 GB)
Rscript 01_ids/scripts/02_inspect_ids.R               # Inspect structure, generate lookup tables
Rscript 01_ids/scripts/03_clean_ids.R                 # Merge and clean all regions
Rscript 01_ids/scripts/04_assign_surveyed_areas.R     # Spatial join: damage areas → survey areas
Rscript 01_ids/scripts/05_compute_area_metrics.R      # Compute area sizes and survey fractions
```

QC scripts in `scripts/qc/` are optional diagnostics — not required to reproduce the cleaned data.
